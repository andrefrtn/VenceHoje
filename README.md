# 💸 VenceHoje

> Controle suas contas, evite atrasos e tenha total domínio sobre seus vencimentos.

---

## 📌 Sobre o projeto

O **VenceHoje** é uma aplicação web desenvolvida para facilitar o controle de **contas, vencimentos e assinaturas recorrentes**.

Com ele, o usuário consegue visualizar todas as suas despesas de forma clara, acompanhar datas importantes e receber uma visão inteligente da sua vida financeira.

O objetivo é simples:  
👉 **nunca mais esquecer um pagamento.**

---

## 🚀 Funcionalidades

- 📊 Dashboard com visão geral das contas
- 📅 Calendário de vencimentos
- 💳 Controle de contas e despesas
- 🔁 Gerenciamento de assinaturas recorrentes
- 📈 Relatórios financeiros
- 🔐 Autenticação de usuário (login/logout)
- 👤 Área de perfil ("Minhas informações")

---

## 🧠 Diferenciais

- Interface moderna e responsiva
- Experiência otimizada para desktop e mobile
- Navegação rápida com React Router
- Sistema de proteção de rotas
- Foco em usabilidade e simplicidade

---

## 🛠️ Tecnologias utilizadas

- React.js 
- React Router DOM
- JavaScript (ES6+)
- CSS3
- LocalStorage (persistência de sessão)
- API REST (backend integrado)
